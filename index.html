What is Java

Java is a popular programming language.

It is used for:
  • Mobile applications
  • Desktop applications
  • Web applications
  • Web servers
  • Games
  • Database connection

Java was developed by Sun Microsystems (which is now a subsidiary of Oracle) in the 1990s.

Example Java code:

System.out.println("Hello, world!");
Install Java

Java may be installed on your PC.

To check if Java is installed, search in the start bar for Java or type the following in Command Prompt (cmd.exe): C:\Users\Your Name\java -version.

If Java is installed, you will se its version information.

Java can be downloaded and installed from the official website.

After installation, you may need to set up the Java environment variables.

Example command to check Java version:

java -version
Variables

Variables are containers for storing data values.

In Java, there are different types of variables.

  • String - store text, such as "Hello".
  • int - stores integers (whole numbers) ex 2.
  • float - stores floating point numbers, with decimals ex 3.14.
  • char - stores single characters, such as 'j' or 'S'.
  • boolean - stores values either true or false.

Variables must be declared before they are used in Java.

Java is a statically-typed language, meaning the type of a variable must be explicitly declared.

Example variable declaration:

int myNum = 10;
Data types

A variable in Java must be a specified data type.

  • int myNum = 6;
  • float myFloatNum = 4.2f;
  • char myLetter = 'E';
  • boolean myBool = true;
  • String myText = "Heya";

Java also supports user-defined data types through classes and interfaces.

Arrays are used to store multiple values of the same type in Java.

Java provides wrapper classes for each primitive data type.

Example array declaration:

int[] myArray = new int[5];
Conditions and If statements

Java supports the usual logical conditions from mathematics.

These conditions can be used in if statements and loops to control the flow of execution.

Java provides several conditional statements, including:

  • if statment
  • if-else statment
  • if-else-if ladder
  • switch statment

Java also supports logical operators such as AND, OR, and NOT.

Conditional expressions can be used to simplify if-else statement.

Example if statement:

if (x > 5) { System.out.println("x is greater than 5"); }
